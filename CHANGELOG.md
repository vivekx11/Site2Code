## [1.0.0] - 2026-03-13

### Added
- Initial release of Site2Code
- Element selection tool with visual highlighting
- HTML extraction with clean formatting
- CSS extraction with computed styles
- Tailwind CSS class generation
- React component generation
- Full page capture functionality
- Project structure generator
- GitHub repository integration
- Download project as ZIP
- Copy to clipboard functionality
- Dark mode and light mode support
- Syntax highlighting with Prism.js
- Settings panel for customization
- Context menu integration
- Keyboard shortcuts support

### Features
- Multi-format code extraction (HTML, CSS, Tailwind, React)
- VS Code-style file explorer
- One-click GitHub push
- Private repository support
- Secure token storage
- Cross-origin stylesheet handling
- Event handler removal from extracted code
- Responsive UI design
- Tab-based navigation
- Real-time element preview

### Technical
- Built with Manifest V3
- Service worker background script
- Content script injection
- Chrome Storage API integration
- GitHub API integration
- JSZip for project packaging
- Prism.js for syntax highlighting

## [Unreleased]

### Known Issues
- Cross-origin stylesheets cannot be fully extracted
- Some dynamically loaded content may not be captured
- Complex JavaScript interactions are not preserved
- Inline SVGs may not format perfectly
