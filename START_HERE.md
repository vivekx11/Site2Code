# 🚀 START HERE - Site2Code Chrome Extension

## Welcome! 👋

You've just received a complete, production-ready Chrome Extension called **Site2Code** that extracts HTML, CSS, and React code from any website.

## ⚡ Quick Start (5 Minutes)

### Step 1: Open the Getting Started Page
Double-click `index.html` to open it in your browser for a visual guide.

### Step 2: Create Icons
1. Open `create-icons.html` in your browser
2. Click "Download All"
3. Move the 3 icon files to the `icons/` folder

### Step 3: Load in Chrome
1. Open Chrome and go to `chrome://extensions/`
2. Turn on "Developer mode" (top right)
3. Click "Load unpacked"
4. Select this folder
5. Done! 🎉

### Step 4: Test It
1. Visit any website (try https://github.com)
2. Click the Site2Code icon in your toolbar
3. Click "Select Element"
4. Hover and click any element
5. See the extracted code!

## 🐛 Troubleshooting

### Extension Won't Load
- Check that icons exist in `icons/` folder
- Verify `manifest.json` is present
- Look for errors in Chrome console

### Element Selection Not Working
- Refresh the webpage after installing
- Check browser console (F12)
- Try a different website

### GitHub Push Fails
- Verify token has `repo` scope
- Check repository name is unique
- Ensure internet connection
