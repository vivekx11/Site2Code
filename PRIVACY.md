# Privacy Policy for Site2Code

**Last Updated: March 13, 2026**

## Overview
Site2Code is committed to protecting your privacy. This extension does not collect, store, or transmit any personal data to external servers.

## Data Collection
We do NOT collect:
- Personal information
- Browsing history
- Website content
- Usage analytics
- Tracking data

## Local Storage
The following data is stored locally in your browser:
- Extension settings (theme, preferences)
- GitHub Personal Access Tokens (encrypted by Chrome)
- User preferences

This data never leaves your device.

## Permissions Explained

### activeTab
Required to access the current webpage's DOM for extracting HTML and CSS when you activate the extension.

### storage
Required to save your preferences and settings locally in Chrome's storage.

### scripting
Required to inject content scripts for element selection and highlighting.

### host_permissions
Required to work on any website you choose to extract code from.

## Third-Party Services

### GitHub API
- Only used when you explicitly connect your GitHub account
- Your Personal Access Token is stored locally
- Only used for creating repositories and pushing code
- Never shared with any other service

## Data Security
- All code extraction happens locally in your browser
- No server-side processing
- Tokens encrypted by Chrome's storage API
- No external analytics or tracking

We may update this policy. Check this page for updates.
