# ⚡ Quick Start Guide

Get Site2Code running in 5 minutes!

## 1️⃣ Generate Icons (2 minutes)

Open `create-icons.html` in your browser:
```bash
# On Windows
start create-icons.html

# On Mac
open create-icons.html

# On Linux
xdg-open create-icons.html
```

1. Choose your colors
2. Click "Download All"
3. Move downloaded files to `icons/` folder

## 2️⃣ Load Extension (2 minutes)

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" (top-right toggle)
3. Click "Load unpacked"
4. Select this folder
5. Pin the extension (click puzzle icon 🧩)

## 3️⃣ Test It (1 minute)

1. Visit any website (try https://github.com)
2. Click the Site2Code icon
3. Click "Select Element"
4. Hover and click any element
5. View the extracted code! 🎉

## Common First-Time Issues

### Icons Not Found?
- Make sure you have 3 files in `icons/` folder:
  - icon16.png
  - icon48.png
  - icon128.png

### Extension Not Working?
- Refresh the webpage after loading the extension
- Check console for errors (F12)

### Can't Select Elements?
- Make sure you clicked "Select Element" button
- Press ESC to cancel and try again
