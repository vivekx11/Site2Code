💻 Development Workflow

Follow these steps for efficient development:

1. Load Extension

1. Open chrome://extensions/ 2. Enable "Developer mode" (top right) 3. Click "Load unpacked" 4. Select this folder 5. Pin the extension

2. Make Changes

# Edit files in your editor # Save changes # Reload extension (click refresh icon) # Test on a webpage

3. Debug

# Popup: Right-click icon → Inspect popup # Background: Extensions page → Service worker → Inspect # Content: F12 on webpage → Console

🐛 Common Issues

  • Extension won't load → Check manifest.json syntax
  • Icons missing → Run create-icons.html
  • Content script not working → Refresh webpage
  • Popup not opening → Check console for errors
