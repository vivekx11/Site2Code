## 🚀 Quick Start (3 Steps)

### 1. Generate Icons
```bash
# Open in browser
open create-icons.html
# Download all icons to icons/ folder
```

### 2. Load Extension
```
1. Go to chrome://extensions/
2. Enable Developer mode
3. Click "Load unpacked"
4. Select this folder
```

### 3. Test It
```
1. Visit any website
2. Click Site2Code icon
3. Click "Select Element"
4. Extract code!
```
